# Kuri Clipboard — クリップボード履歴＆定型文管理ツール

**軽量・高速・インストール不要** の常駐型クリップボードマネージャー（Windows 10 / 11 対応）

</div>

---

## 概要

コピーしたテキストを自動で履歴に残し、よく使う文章を「定型文」として登録してワンクリックで貼り付けられる常駐ツールです。
Rust + Windows API のみで実装しており、**単一の実行ファイル（約 290 KB）・外部 DLL 不要・インストール不要**で動作します。タスクトレイに常駐し、CPU・メモリの消費はごくわずかです。

## 特長

- 🪶 **超軽量** — 単一 EXE 約 290 KB、常駐メモリ約 8 MB、CPU 使用ほぼゼロ（イベント駆動でポーリングなし）
- 📋 **自動履歴** — コピーした瞬間に履歴へ記録。重複は自動で除外
- ⭐ **定型文** — 定型文を登録してワンクリック貼り付け。履歴からそのまま定型文化も可能
- ⌨️ **豊富なショートカット** — 履歴の直接貼り付けなどをグローバルホットキーで実行。すべて任意のキーに変更可能
- 🌗 **ダークモード対応** — OS のライト／ダークテーマに自動追従
- 🖱️ **賢い貼り付け** — 直前まで使っていたウィンドウのカーソル位置へ正確に貼り付け
- 💾 **ポータブル** — 設定・履歴は EXE と同じフォルダに保存。USB メモリ等で持ち運び可能

## 動作環境

- Windows 10 / 11（64bit）

## インストール

インストール不要です。`clipboard_manager.exe` を任意のフォルダに置いて実行するだけです。
アンインストールは EXE と設定ファイルを削除するだけで完了します（レジストリは使用しません）。

## 使い方

### 起動

実行するとウィンドウが表示され、同時にタスクトレイに常駐します。
ウィンドウを閉じても常駐は続き、トレイアイコンからいつでも再表示できます。
二重起動は自動的に抑制されます（2 回目の起動は既存ウィンドウを前面に表示します）。

### 基本操作

| 操作 | 動作 |
| --- | --- |
| トレイアイコン左クリック | ウィンドウを表示 |
| トレイアイコン右クリック | メニュー（表示／終了） |
| 項目をダブルクリック / Enter | 直前まで使っていたウィンドウへ貼り付け |
| Delete キー | 選択中の項目を削除 |
| Esc キー / 閉じるボタン | ウィンドウを隠す（常駐は継続） |

- ウィンドウは貼り付けや設定などの操作後も閉じません
- 通常のウィンドウとして最小化・最大化・リサイズができます
- 左上の **「モード」ボタン** を押すたびに「履歴」⇔「定型文」が切り替わります
- ウィンドウの初期位置は画面右側の中段。ドラッグで移動でき、以降はその位置を維持します

### 履歴モード

- テキストをコピーすると自動で履歴に追加されます（最新が先頭、番号付き表示）
- 履歴内に同じ内容が既にある場合は重複登録しません（初期設定では既存項目を先頭へ移動）
- **貼り付け** / **定型文に登録**（履歴モードのみ表示）/ **削除** / **設定**

### 定型文モード

- **貼り付け** / **追加**（複数行可）/ **編集** / **削除** / **設定**

### ショートカットキー（初期値）

| アクション | 初期キー |
| --- | --- |
| クリップボードを定型文に登録 | `Ctrl + Alt + V` |
| 履歴 1〜9 を直接貼り付け | `Ctrl + Alt + 1` 〜 `Ctrl + Alt + 9` |
| 履歴 1〜9 を消去 | （なし） |
| 定型文 1〜9 を貼り付け | （なし） |
| ウィンドウ表示／非表示 | （なし） |
| 常に手前に表示の切り替え | （なし） |

すべてのアクションは設定画面から任意のキーに変更・解除できます。
他のアプリが既に使用しているキーは登録できず、その場合は通知されます。

> **補足:** 履歴貼り付けの初期値を `Ctrl + Shift + 1〜9` にしていないのは、`Ctrl + Shift + 1〜5` が
> Excel のセル書式ショートカットと衝突するためです。Excel を使わない場合は設定で変更できます。

### 設定

- 履歴・定型文の保持数（各 1〜1000、初期値 10）
- 項目のフォントサイズ（6〜40pt、初期値 10）
- ウィンドウを常に手前に表示（初期値 オン）
- 同じ内容を再コピーしたら履歴の先頭へ移動（初期値 オン）
- ショートカットキーの割り当て・解除

## データの保存場所

履歴・定型文・各種設定は、EXE と同じフォルダの **`clipboard_manager_data.txt`**（プレーンテキスト）に
自動保存され、次回起動時に復元されます。フォルダごとコピーすれば設定ごと別の PC に持ち運べます。

> ⚠️ このファイルにはコピーした内容が平文で保存されます。パスワードなど機微な情報を扱った場合は、
> 必要に応じて履歴を削除するか、ファイルごと削除してください。

## ソースからのビルド

Rust（GNU または MSVC ツールチェーン）が必要です。

```sh
cargo build --release
```

生成物: `target/release/clipboard_manager.exe`

- 依存クレートは `windows-sys`（Windows API バインディング）のみ
- MSVC は不要で、GNU ツールチェーン + MinGW でもビルドできます
- アプリアイコンは付属の `generate_icon.ps1` で生成し、ビルド時に `windres` で EXE へ埋め込みます

## 技術メモ

- クリップボード監視は `AddClipboardFormatListener` によるイベント駆動（ポーリングなし）
- GUI フレームワークは未使用。Windows 標準コモンコントロールを直接利用
- 貼り付けは「クリップボードへ設定 → 直前のウィンドウへフォーカスを戻す → `Ctrl + V` 送出」方式
- 貼り付け先は WinEvent フックでフォアグラウンドの変化を常時追跡して特定
- ダークモードは DWM のダークタイトルバー + テーマ + 独自配色（文字が背景に同化しない配色）で実装し、OS のテーマ変更に動的追従
